# Expense Tracker

A simple React-based expense tracker that allows users to record and monitor their expenses in real-time.

## 🚀 Features

- Add and remove expenses
- Dynamic balance and totals
- Simple, clean UI
- React hooks and component-based structure

## 🛠️ Built With

- React.js
- JavaScript (ES6+)
- CSS Modules
