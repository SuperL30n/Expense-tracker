# 💸 Expense Tracker App

A simple and responsive React app that helps users track their daily expenses. It supports real-time tracking, category-based summaries, and data visualization with a pie chart.

## 🔗 Live Demo

👉 [View Live](https://expense-tracker-ashy-sigma.vercel.app/)

---

## 🧰 Tech Stack

- **React**
- **CSS Modules**
- **JavaScript (ES6+)**
- **Recharts (for charts)**
- **Vercel** (deployment)

---

## 🚀 Features

- Add new expenses with amount, description, and category
- Real-time calculation of total spending
- Delete individual expense items
- Visual summary of expenses by category using a pie chart
- Clean and responsive UI
- Input validation and inline error handling

## 🛠 How to Run Locally

```bash
# Clone the repo
git clone https://github.com/SuperL30n/Expense-tracker.git

# Navigate into the project folder
cd Expense-tracker

# Install dependencies
npm install

# Start the development server
npm run dev
```
